# COVIDeogame
## \*NIX / Linux
### Instalación
```console
foo@bar:/path/to/repo$ source ./venv/sh/bat
```
### Ejecución
```console
(CIIE2021-2D-EC1) foo@bar:/path/to/repo$ ./main.py
```
### Tras la ejecución, cerrar el terminal o ejecutar
```console
(CIIE2021-2D-EC1) foo@bar:/path/to/repo$ deactivate
```
